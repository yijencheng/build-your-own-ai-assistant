# First Agent (Gemini, Python)

## Setup

```bash
pip install google-genai
export GEMINI_API_KEY="your_api_key"
# optional:
# export GEMINI_MODEL="gemini-2.0-flash"
```

## Run basic chat agent

```bash
python3 '1 - Building Our First Agent/agent.py'
```

Type `q` or `quit` to exit.

## Run read_file tool demo

```bash
python3 '1 - Building Our First Agent/agent_with_read_file_tool.py'
```

Ask it to read files, for example: `Read README.md`.
